# User Stories

## Sales Manager

As a Sales Manager,

I want to ask:

"What are total sales?"

So that I can quickly understand business performance.

---

## Team Lead

As a Team Lead,

I want to ask:

"What is the top-performing product?"

So that I can focus resources appropriately.

---

## Business User

As a Business User,

I want information without navigating reports,

So that I can make decisions more efficiently.

---

## Future User Story

As an Executive,

I want to ask questions through Teams,

So that I can obtain business insights without opening reporting tools.
